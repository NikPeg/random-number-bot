# random-number-bot

Телеграм бот для генерации случайных чисел.

## Команды

| Команда | Результат | Триггер-слова |
|---|---|---|
| `/double` | Случайное 2-значное число (10–99) | дабл |
| `/triple` | Случайное 3-значное число (100–999) | трипл |
| `/quadruple` | Случайное 4-значное число (1000–9999) | квадрипл, четырипл |
| `/quintuple` | Случайное 5-значное число (10000–99999) | квинтипл, пентипл, пятерипл |
| `/random` | Случайное число от 0 до 1 000 000 | |
| `/random 500` | Случайное число от 0 до 500 | |
| `/random 10 20` | Случайное число от 10 до 20 | |
| `/alice` | Случайный стикер | |
| `/coffee` | Кофя | кофе |
| `/hurry` | Черемша | спеша |
| `/who` | Случайный участник чата (только если бот — админ) | кто |
| `/stats` | Статистика чатов (только для администратора бота) | |
| `/how` | Как? | как?, СССР, советский союз, в советском союзе |
| `/help` | Список команд | |

Триггер-слова регистронезависимы — бот реагирует на них в любом сообщении.

## Переменные окружения

| Переменная | Описание | По умолчанию |
|---|---|---|
| `BOT_TOKEN` | Токен Telegram бота | — |
| `ALICE_STICKER_PACK` | Название стикерпака для `/alice` | `inthenameofAlice` |
| `ADMIN_ID` | Telegram ID администратора (для `/stats`) | — |
| `DELETE_CHANNEL_MESSAGES` | Удалять сообщения от имени каналов | `false` |

### DELETE_CHANNEL_MESSAGES

Если установлено в `true`, бот автоматически удаляет все сообщения, отправленные от имени канала или анонимно (не от реального пользователя). Требует прав администратора в чате. Если бот не админ — молча игнорирует.

## Запуск через Docker Compose

1. Скопируй `.env.example` в `.env` и заполни переменные:

```bash
cp .env.example .env
# отредактируй .env
```

2. Запусти:

```bash
docker compose up -d
```

3. Логи:

```bash
docker compose logs -f
```

4. Остановить:

```bash
docker compose down
```
