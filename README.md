# random-number-bot

Телеграм бот для генерации случайных чисел.

## Команды

| Команда | Результат |
|---|---|
| `/triple` | Случайное 3-значное число (100–999) |
| `/quadruple` | Случайное 4-значное число (1000–9999) |
| `/quintuple` | Случайное 5-значное число (10000–99999) |
| `/random` | Случайное число от 0 до 1 000 000 |
| `/random 500` | Случайное число от 0 до 500 |
| `/random 10 20` | Случайное число от 10 до 20 |
| `/help` | Список команд |

## Запуск через Docker Compose

1. Скопируй `.env.example` в `.env` и вставь токен бота:

```bash
cp .env.example .env
# отредактируй .env, укажи BOT_TOKEN
```

2. Запусти:

```bash
docker compose up -d
```

3. Логи:

```bash
docker compose logs -f
```

4. Остановить:

```bash
docker compose down
```
